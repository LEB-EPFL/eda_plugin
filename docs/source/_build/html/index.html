Welcome to event driven acquisition!¶

Note

This documentation is a work in progress.

Event-driven-acquisition (EDA) is a Python library is a Python library for advanced microscope control routines to enable acquisitions to react to specific biological events of interest.

More information on the project it was first used on can be found in the bioRxiv article.

Check out the Usage section for further information, including how to Installation the project.

Note

This project is under active development.
