eda_plugin.actuators.daq.EDAAcquisition¶

class eda_plugin.actuators.daq.EDAAcquisition(*args: Any, **kwargs: Any)¶

Bases: PyQt5.QtCore.

An acquisition that can be run from the main Actuator instance.

This setup of a separate object for the Acquisition comes from the isimgui, where there are two different classes for the live mode and the mda acquisition. Here it could be included in the main class, but to keep things consistent it will remain in its own class.
