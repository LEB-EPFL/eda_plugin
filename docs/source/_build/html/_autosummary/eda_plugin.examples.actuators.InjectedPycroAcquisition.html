eda_plugin.examples.actuators.InjectedPycroAcquisition¶

class eda_plugin.examples.actuators.InjectedPycroAcquisition(*args: Any, **kwargs: Any)¶

Bases: PyQt5.QtCore.

Acquisition based on pycromanager with image injection.

This can be used with the DemoConfig of MicroManager. It injects an image from a tif-stack to replace the image from the DemoCam. This allows to test the performance of the EDA loop and the model used without a sample or even a microscope.

receive_image(image: numpy.ndarray, metadata)¶

Overwrite the function in the original PycroAcquisition.

Add the image replacement before calling the original function on the injected image. The frame is chosen from the tif stack according to the time passed since the last acquired frame.
