Bases: object

Settings mainly as in MM, but some things are iSIM specific.

Sweeps_per_frame: How many times does the galvo go back and forth during one exposure. Can be bigger for longer exposure times, to reduce the time a specific part is exposed to light, to reduce phototoxicity.
