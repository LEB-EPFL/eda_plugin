eda_plugin.analysers.image module¶

Basic analyser implementation for images.

This basic analyser is set up to receive images from the EventBus, analyse them using basic image analysis functions and send a decision parameter back to the EventBus to be handed on to an interpreter.

class eda_plugin.analysers.image.ImageAnalyser(*args: Any, **kwargs: Any)¶

Bases: PyQt5.QtCore.

Basic image analyser.

Receives and gathers the needed amount of images as set in settings.json. Once all are received it tries to start a worker in a threadpool to analyse the image. The worker itself will pass the value it calculated on to any expecting interpreters.
